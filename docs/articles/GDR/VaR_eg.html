Introduction

Nous allons ici nous int√©resser aux applications de la Value at Risk (VaR) en finance. La VaR est une mesure de risque qui permet d‚Äôestimer les pertes maximales potentielles d‚Äôun portefeuille d‚Äôactifs financiers sur un horizon de temps donn√©, √† un certain niveau de confiance. Elle est largement utilis√©e par les institutions financi√®res pour √©valuer et g√©rer les risques de march√©, de cr√©dit et de liquidit√© (cf. Value at-Risk)

Test de stationarit√© du log-rendement

Pour utiliser le log rendement comme variable profit et perte (PnL : Profit and Loss) pour impl√©ter la VaR, nous devons tester sa stationarit√©. En effet, la stationarit√© est une propri√©t√© importante des s√©ries temporelles financi√®res. Une s√©rie temporelle est dite stationnaire si ses propri√©t√©s statistiques telles que la moyenne, la variance et la covariance restent constantes au fil du temps.

Pour plus de certitude, nous allons effectuer un test de Dickey-Fuller augment√© (ADF) pour tester la stationnaire dans la s√©rie :

\[ H_0=\rho = 1, \alpha=0 \]

Dans le cas d‚Äôune s√©rie AR(1)(\(X_t = \alpha + \rho X_{t-1} + \xi_t\)) avec intercept, la s√©rie est non stationnaire si \(\rho = 1\) et stationnaire si $< 1. Dans le cas contraire, il faudrait consid√©rer la s√©rie de diff√©rences pour la rendre stationnaire.

Ainsi la s√©rie des log-rendements est stationnaire. Nous pouvons donc utiliser les rendements logarithmiques pour calculer la VaR √† horizon 1 jour.

VaR analytique

Pour rappel, la VaR analytique ou gaussienne est bas√©e sur la distribution gaussienne des rendements. Nous allons utiliser la distribution normale pour calculer la VaR √† horizon 1 jour. La VaR √† horizon 1 jour est d√©finie comme suit :

\[ VaR = -\mu_{PnL} + \Phi^{-1}(\alpha) \times \sigma_{PnL} \] o√π \(\Phi^{-1}(\alpha)\) est le quantile de la distribution normale du PnL (Profit and Loss) √† \(\alpha\).

from scipy import stats
def gaussian_var(PnL, seuil):
    mean_PnL = np.mean(PnL)
    sd_PnL = np.std(PnL)
    VaR = - mean_PnL + sd_PnL * stats.norm.ppf(seuil)
    return VaR

seuil = 0.99
prices= train_close["Close"].iloc[-1]*(1+train_close["Return"])
train_close["PnL"] = prices - train_close["Close"].iloc[-1]


VaR_gaussienne = gaussian_var(train_close["PnL"], seuil)

VaR Monte Carlo

La VaR Monte Carlo est bas√©e sur la simulation de trajectoires de rendements. Nous allons simuler 1000 trajectoires de rendements logarithmiques et calculer la VaR √† horizon 1 jour en supposant que le prix de la cl√¥ture suit une distribution normale.

def monte_carlo_var(Close,Return,seuil, num_simulations):
  mean = np.mean(Return)
  std = np.std(Return)
  # Generate random scenarios of future returns
  simulated_returns = np.random.normal(mean, std, size=num_simulations)
  simulated_prices = Close.iloc[-1] * (1+simulated_returns)
  simulated_pnl = simulated_prices - Close.iloc[-1]

  # Calculate portfolio values for each scenario
    
  return historical_var(simulated_pnl,seuil)

def bootstrap_var(PnL, seuil, num_simulations, alpha_IC, B):
    VaRs_boot = np.zeros(num_simulations)

    for i in range(num_simulations):
        sample = np.random.choice(PnL, B, replace=True)
        VaRs_boot[i] = historical_var(sample, seuil)

    VaR = np.mean(VaRs_boot)

    lower_bound = np.percentile(VaRs_boot, 100 * (1-alpha_IC) / 2)
    upper_bound = np.percentile(VaRs_boot, 100 * (1 - (1-alpha_IC) / 2))
    IC = (lower_bound, upper_bound)

    return VaR, IC
